Ans: JavaScript is a single-threaded programming language. It means , Js works like it can handle one task at a time. And it uses dynamic typing, where type safety is verified at the runtime. These days there are two flavors of javascript. On the one hand: jS is in the browser. In here you get a bunch utilities to ,modify your html document, making network request, managing data and many many more. On the other hand: JS on the server. There you got a very different set of utilities, that let you manage the computer directly, access files and folders etc. In many aspects, JS on the server is close to what other programming language usually do.
